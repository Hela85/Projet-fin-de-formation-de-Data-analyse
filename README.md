# Projet_de_fin_de_formation_de_Data-analyse

 ## Analyse et prédiction de l'abandon du e-learning dans les plateformes numériques

L'apprentissage en ligne connait un essort considérable. En 2024, 860millions de personnes étaient inscrites à des cours en ligne. Derrière ce chiffre impressionnat se cache un paradoxe : un taux d'abandon extrêmement élevé pouvant atteindre 90% ! 
Comprendre ce phénomène et pouvoir l'anticiper est un enjeu majeur, aussi bien pour les plateformes éducatives que pour tout autre secteur où le désengagement et le churn est un véritable problème comme dans la santé, finance, énergie ou encore la e-santé!
Comment expliquer cet abandon et surtout comment le prédire pour mieux le prévenir ?!
